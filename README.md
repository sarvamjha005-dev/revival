Introduction :
AI-based Resume Analyser that uses Natural langauge Processing Techniques to determine candidate's elgibility based upon on 
Pragmatic score and Keyphrase Score of  their CV/Resume document.It Also returns The Ranking of multiple candidates based on their CV/resume.The Application uses FastAPI endpoints ,HTML U.I and has been
containerized using dockers .The appplication has been deployed on Render.

Components:-
main.py:-Resume Analyser code
indexy.html-User interface code
Dockerfile:Dockerfile for the the application
.gitignore :-used by dockerfile to ignore certain files
requirement.txt:-files needed by docker
